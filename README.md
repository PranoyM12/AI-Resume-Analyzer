# AI Resume Analyzer

An AI-powered resume analysis tool that evaluates how well a resume matches a job description using NLP and semantic similarity.

## Live Demo
[https://your-app-name.streamlit.app](https://ai-resume-analyzer-iynmfgxwtvbpb6zvvstsem.streamlit.app/)

## Features

- Resume parsing from PDF
- Skill extraction using NLP
- Semantic job matching
- AI-generated resume improvement suggestions
- Resume skill visualization (Word Cloud)

## Tech Stack

Python  
Streamlit  
Sentence Transformers  
HuggingFace Transformers  
Scikit-learn  

## How It Works

1. Upload resume
2. Paste job description
3. AI extracts skills
4. Semantic similarity calculates match score
5. AI generates improvement suggestions

## Run Locally

pip install -r requirements.txt  
streamlit run app.py
